# 황금레시피 웹사이트 Blueprint

## 프로젝트 개요

이 프로젝트는 사용자가 다양한 한식 레시피를 쉽게 찾아보고 따라 할 수 있도록 돕는 간단한 웹 애플리케이션입니다. 각 레시피는 별도의 페이지에 상세한 재료와 조리 과정을 안내하며, 메인 페이지에서는 전체 레시피 목록을 아이콘과 함께 보여줍니다.

## 구현된 기능 및 디자인

* **메인 페이지 (index.html):**
    * 전체 레시피 목록을 그리드 형태로 보여줍니다.
    * 각 레시피 카드는 아이콘, 이름, 간단한 설명으로 구성됩니다.
    * 레시피 카드를 클릭하면 해당 레시피의 상세 페이지로 이동합니다.
* **레시피 상세 페이지 (예: kimchi-jjigae.html):**
    * 레시피 이름, 아이콘, 제공량을 표시합니다.
    * 필요한 재료 목록을 보여줍니다.
    * 조리 과정을 단계별로 아이콘과 함께 설명합니다.
    * 각 조리 단계마다 예상 소요 시간을 표시합니다.
    * 메인 페이지로 돌아갈 수 있는 '돌아가기' 버튼이 있습니다.
* **추가된 레시피:**
    * 등갈비 김치찜
* **디자인 및 스타일링 (style.css, recipe-detail.css):**
    * 따뜻하고 부드러운 색감 (#fdf6e3, #d2691e, #a0522d)을 사용하여 편안한 느낌을 줍니다.
    * 카드형 UI와 그리드 레이아웃을 사용하여 정보를 명확하게 전달합니다.
    * 사용자가 쉽게 상호작용할 수 있도록 버튼과 링크에 hover 효과를 적용했습니다.
    * 모든 페이지에 일관된 디자인을 적용하여 통일성을 높였습니다.

## 현재 계획 및 단계

1. **Git 저장소 초기화:** 로컬 프로젝트를 Git 저장소로 만듭니다.
2. **GitHub 연동:** 로컬 저장소를 원격 GitHub 저장소와 연결합니다.
3. **코드 푸시:** 현재까지 작업한 모든 파일을 GitHub에 업로드합니다.
4. **Firebase 호스팅 설정:** Firebase를 사용하여 웹사이트를 배포할 준비를 합니다.
5. **배포:** Firebase 호스팅을 통해 웹사이트를 공개적으로 배포합니다.
